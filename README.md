# Nike-Social-Media-Analytics-PowerBI
Interactive Power BI dashboard for social media analytics with KPI tracking, sentiment analysis, funnel insights, and competitor comparison.

🔹 Project Overview

This project presents a comprehensive Social Media Analytics Dashboard built using Power BI to analyze brand performance across multiple platforms. It combines KPI tracking, sentiment analysis, marketing funnel insights, and competitor benchmarking into a fully interactive dashboard.

🎯 Objectives
📌 1:
Build a dashboard with:
KPI Cards (Reach, Likes, Followers, CTR)
Sentiment Analysis (Pie Chart)
Monthly Engagement Trends (Line Chart)
Geographic Reach (Map)
Platform-based filtering

📌 2:
Develop an advanced dynamic KPI dashboard with:
Custom DAX measures
Date range filtering
Funnel analysis
Competitor comparison
Drill-through analysis

⚙️ Tools & Technologies

Power BI Desktop
DAX (Data Analysis Expressions)
Power Query (Data Cleaning)
Excel Dataset
📁 Dataset Description - The dataset consists of 4 main tables:

SocialMediaPosts – Post-level engagement data (likes, comments, reach, sentiment, etc.)
CompetitorAnalysis – Brand-wise metrics across platforms
KPI_Summary – Monthly aggregated KPIs
Funnel_Data – Marketing funnel stages and user drop-off

📊 Key Insights

🔹 Engagement Insights
Facebook & Twitter show highest engagement rates (~12.7–12.8%)
YouTube has highest reach but lower engagement (passive audience)
🔹 Sentiment Analysis
68% Positive sentiment
19% Negative → requires attention
13% Neutral → opportunity for conversion
🔹 Funnel Analysis
Major drop-off at Reach → Engagement (84%)
Indicates need for better content & CTA strategies
🔹 Geographic Insights
Delhi has highest reach (5.1M)
Mumbai shows lower engagement than expected
🔹 Competitor Insights
Higher ad spend ≠ higher engagement
Content quality & targeting are key differentiators
